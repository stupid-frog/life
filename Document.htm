alert方法

默认调用：alert('请打开麦克风')

回调函数：alert('请打开麦克风').then(callback)

支持多参数：alert({ title: '我是标题', content: '请打开麦克风', doneText: '按钮文字' }).then(callback)


confirm方法

默认调用：confirm('请打开麦克风')

confirm({ title: '我是标题', content: '请打开麦克风', doneText: '确认按钮文字', cancalText: '取消按钮文字' }).then(() => { console.log('已确认') }).catch(() => { console.log('已取消') })


toast方法

默认调用：toast('请打开麦克风')

设置停留时间：toast({time: 5000, content: '停留5秒再消失'})
窗口尺寸: 1600 x 761
视窗尺寸: 1600 x 761
